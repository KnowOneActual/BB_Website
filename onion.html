About This Onion Service

This site is an experiment in privacy-first web development, built specifically to run as a Tor Onion Service. It is intentionally simple, lightweight, and avoids all JavaScript, tracking, and third-party resources.

The primary goal of this project was to learn the process of setting up a hidden service from scratch on a cloud server. It demonstrates proficiency in Linux server administration (Ubuntu), web server configuration (Nginx), and networking (Tor). Many features from my main website—like the weather app, analytics, and interactive elements—have been removed by design, as they are incompatible with the goal of a truly private and secure browsing experience.

For the most up-to-date information, please visit my main website: beaubremer.com.
