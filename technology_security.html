Technology & Security

A transparent look at the stack and security posture of this site.

The Tech Stack

Frontend

  • Core: Semantic HTML5, CSS3, JavaScript (ES6+)
  • Styling: Tailwind CSS 3.4 (Local Build)
  • Typography: Space Grotesk & Inter
  • Icons: Font Awesome 7.0 & Feather Icons

Infrastructure

  • Hosting: Netlify
  • DNS/WAF: Cloudflare
  • Functions: Netlify Serverless (Node.js)
  • Analysis: ESLint & Snyk Security

Security Posture

Content Security Policy (CSP)

Implemented a strict CSP to prevent XSS and data injection attacks. We have externalized all JavaScript to remove the need for 'unsafe-inline' execution, significantly hardening the site.

Privacy-First Typography

All fonts are self-hosted locally. This prevents third-party tracking from font providers and ensures the site renders correctly even for users with strict privacy settings or fingerprinting protection enabled.

Static Analysis & Linting

Using ESLint with security-focused plugins to scan the codebase for logic bugs and potential security risks during development.

Automated Vulnerability Scanning

Integrated Snyk and GitHub Dependabot for real-time vulnerability scanning of dependencies and infrastructure as code.

For more details or to report a vulnerability, please see the Security Acknowledgements page.
