Project Status

This site is a placeholder and a proof of concept.

It is not actively maintained and may not always be online. For the most up-to-date information about me, please visit my main website at beaubremer.com.

About This Site

This site is an experiment in privacy-first web design, built specifically for the Tor network. It is intentionally simple, lightweight, and does not use JavaScript or third-party resources.

The primary goal of this project was to learn the process of setting up a Tor onion service from scratch on a cloud server. It serves as a successful demonstration of that experiment. More content and privacy-focused tools may be added over time.

Many features from my main website—like the weather app, analytics, and interactive elements—have been stripped away. This is not because they are broken, but because they are incompatible with the goal of a truly private and secure browsing experience. Features that rely on knowing your location or running complex scripts could compromise your anonymity, so they have been removed by design.

What's left is pure, simple content.
